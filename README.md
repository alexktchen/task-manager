# task-manager

#### Run docker compose

```
docker-compose up -d
```

# task-manager API Doc

### local domain: 127.0.0.1:8080

## End-point: /api/v1/task

#### Add new task

### Method: POST

> ```
> {{domain}}/api/v1/task
> ```

### Body (**raw**)

```json
{ "name": "test" }
```

## End-point: /api/v1/task/:id

#### Update task by id

### Method: PUT

> ```
> {{domain}}/api/v1/task/:id
> ```

### Body (**raw**)

```json
{
  "id": 5,
  "name": "1111",
  "status": 1
}
```
